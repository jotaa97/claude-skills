# Crear carrusel de LinkedIn (formato #1 por alcance)

El carrusel PDF es el formato que más engagement genera en LinkedIn (≈6x el texto solo, ≈3x el
vídeo, según el análisis de 2M+ posts de Buffer) porque maximiza el *dwell time*. Cuando el usuario
quiera un carrusel, genera el contenido **slide a slide** listo para maquetar. Español de España.

## Spec del carrusel (síguela siempre)
1. **Portada:** un GANCHO potente (1 frase que para el scroll: resultado o dolor) + promesa clara.
   Es la miniatura: decide si deslizan. Añade «Desliza →».
2. **Problema:** el dolor del lector en 1 frase grande.
3. **Pasos (4-6 slides):** UNA SOLA IDEA por slide (título corto + 1 línea de detalle). Mezcla
   «qué se automatiza» + «cómo se monta con Claude Code» + «lo que gana el negocio» en una narrativa.
4. **Resumen:** recap + invitación a guardar («Guárdalo si…»).
5. **CTA:** quién eres + llamada suave (entender su tarea, escribir por privado). El enlace/recurso
   va en el **primer comentario**, nunca dentro del carrusel.

Longitudes para que quepa en el slide: gancho ≤85 car., promesa ≤110, problema ≤120, título de paso
≤38, detalle ≤115, resumen ≤180, CTA ≤180.

## Reglas de oro
1. **Una idea por slide.** Texto grande, legible en móvil. Nada de párrafos.
2. **PROHIBIDO el engagement-bait** («comenta PALABRA y te lo envío»): penaliza el alcance.
3. **No inventes cifras ni casos.** Si falta un dato real, escribe `[DATO REAL]`.
4. **Herramientas: solo Claude / Claude Code / Claude Design.**
5. **Empieza por el lector** (su problema), no por ti.
6. **Formato:** documento PDF vertical **1080×1350 (4:5)**, 7-10 slides. Paleta sobria del usuario.

## Cómo actuar
1. Pregunta breve si falta: tema/sector, y si hay algún dato o caso real.
2. Entrega el carrusel **slide a slide** (numerado), listo para maquetar en Canva/Claude Design o
   para un generador PDF.
3. Añade debajo: el **post de acompañamiento** (gancho + por qué importa + «te lo dejo en el
   carrusel» + pregunta final) y la **idea de imagen de portada** (casera/diagrama, no flyer).
4. Recuerda: publicar en hora dorada (mañana L-V), responder los comentarios la primera hora.

## Contexto del usuario (por defecto)
Automatización con IA para pymes montada con **Claude Code**; método propio «automatización con
sentido» (arregla el proceso antes de automatizar); construye en público.
